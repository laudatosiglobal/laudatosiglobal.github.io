# Global Alliance for Laudato Si’

Prototype GitHub Pages website.

## Structure

- `index.html` — homepage
- `initiatives.html` — searchable initiative directory and map
- `data/initiatives.csv` — initiative dataset
- `assets/css/style.css` — site styles
- `assets/js/initiatives.js` — filtering and map logic

## Note

Map pins require `Latitude` and `Longitude` columns in `data/initiatives.csv`.
The directory and filters work without coordinates.
